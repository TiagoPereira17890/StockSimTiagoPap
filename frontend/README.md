# React + Vite

Este template fornece uma configuração mínima para que o React funcione no Vite com HMR e algumas regras do ESLint.

Atualmente, estão disponíveis dois plugins oficiais:

- [@vitejs/plugin-react](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react) usa [Oxc](https://oxc.rs)
- [@vitejs/plugin-react-swc](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react-swc) usa [SWC](https://swc.rs/)

## React Compiler

O React Compiler não está ativado neste template devido ao seu impacto no desempenho durante o desenvolvimento e a compilação. Para adicioná-lo, consulte [esta documentação](https://react.dev/learn/react-compiler/installation).

## Expandir a configuração do ESLint

Se estiver a desenvolver uma aplicação para produção, recomendamos que use TypeScript com as regras de lint baseadas em tipos ativadas. Verifique o [template TS](https://github.com/vitejs/vite/tree/main/packages/create-vite/template-react-ts) para obter informações sobre como integrar o TypeScript e o [`typescript-eslint`](https://typescript-eslint.io) no seu projeto.
